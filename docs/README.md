## 快速链接

### 开机自启

Windows 和 Linux 平台的开机自启功能，支持 UI 版本和 CMD 版本。可通过脚本或应用内设置管理。

### Machine ID

跨平台的机器唯一标识符获取模块，支持 Windows/Linux/macOS，并提供备用方案。
